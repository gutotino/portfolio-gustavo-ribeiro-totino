# 🧩 Engenharia Reversa com IA

Projeto acadêmico desenvolvido com foco em engenharia reversa assistida por Inteligência Artificial, analisando os impactos do uso de ferramentas generativas no desenvolvimento de software.

## 📌 Objetivo

Refletir sobre o uso de IA na engenharia de software, explorando:

- Engenharia reversa
- Desenvolvimento assistido por IA
- Ética no uso de ferramentas generativas
- Pensamento lógico e crítico
- Prompt Engineering

## 🛠️ Tecnologias e Conceitos

- IA Generativa
- Engenharia Reversa
- Prompt Engineering
- HTML/CSS
- UX/UI

## 💡 Reflexões Desenvolvidas

O projeto aborda competências indispensáveis para profissionais da área de tecnologia diante da evolução das IAs, como:

- Capacidade analítica
- Interpretação crítica de código
- Comunicação clara com IA
- Ética no desenvolvimento

Também discute o limite entre aprendizado e plágio digital no uso de ferramentas generativas.
