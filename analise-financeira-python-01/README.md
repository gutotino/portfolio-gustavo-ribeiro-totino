# 📈 Análise Financeira com Python

## 📝 Descrição

Projeto acadêmico desenvolvido para análise de dados financeiros utilizando Python.

O sistema realiza processamento de informações de ativos financeiros, permitindo visualizar preços, volume de negociações e simulações de carteira de investimentos através de gráficos e análises estatísticas.

---

## 🚀 Funcionalidades

- Leitura e tratamento de dados financeiros
- Simulação de carteira de investimentos
- Visualização gráfica de ativos
- Análise de desempenho financeiro
- Geração de relatórios e gráficos

---

## 🛠 Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab
- Jupyter Notebook

## 📊 Resultados e Aprendizados

Durante o desenvolvimento do projeto foram aplicados conceitos de:

- Interpretação de indicadores financeiros
