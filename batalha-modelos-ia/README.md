# ⚔️ Batalha de Modelos de IA

Projeto desenvolvido na disciplina de Engenharia de Prompt e Aplicações em IA com foco na comparação entre diferentes modelos de Inteligência Artificial utilizando prompts estruturados em XML.

## 📌 Objetivo

Construir um prompt estruturado para geração de uma página HTML Single Page com CSS integrado e testar o desempenho em diferentes IAs generativas, analisando:

- Precisão dos resultados
- Qualidade do HTML
- Criatividade
- Bugs e erros de sintaxe
- Consumo de tokens

## 🧠 Modelos Avaliados

- ChatGPT
- Gemini
- DeepSeek
- Qwen
- Grok
- Maritaca
- Claude

## 🛠️ Tecnologias e Conceitos

- Engenharia de Prompt
- XML
- HTML5
- CSS3
- IA Generativa
- UI/UX

## 📊 Resultados

O projeto apresentou uma análise comparativa entre os modelos de IA, destacando diferenças de desempenho, criatividade, responsividade e capacidade de interpretação de prompts complexos.
