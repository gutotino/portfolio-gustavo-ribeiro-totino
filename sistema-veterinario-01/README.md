# 🐾 Sistema Veterinário

## 📝 Descrição

Projeto acadêmico desenvolvido para modelagem e estruturação de um sistema de gerenciamento para clínica veterinária.

O sistema foi planejado para organizar informações de tutores, pets, consultas, prontuários e atendimentos realizados na clínica.

---

## 🚀 Funcionalidades Modeladas

- Cadastro de tutores
- Cadastro de pets
- Controle de consultas
- Registro de prontuários
- Relacionamento entre entidades

---

## 🗄️ Modelagem de Banco de Dados

O projeto inclui:
- Diagrama Entidade-Relacionamento (ER)
- Estrutura lógica do banco de dados
- Relacionamentos entre tabelas
- Organização de atributos e cardinalidades
