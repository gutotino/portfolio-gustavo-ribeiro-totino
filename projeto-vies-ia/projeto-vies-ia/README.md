# 🤖 Classificação de Dados com Teachable Machine

## 📝 Descrição da Atividade
Este projeto foi desenvolvido como uma atividade prática de introdução à **Inteligência Artificial**. Utilizando a ferramenta **Teachable Machine** (Google), realizei o treinamento de um modelo de aprendizado de máquina supervisionado para reconhecimento de padrões.

O objetivo foi entender o fluxo de trabalho de uma IA: coleta de dados, treinamento do modelo e teste de acurácia.

> 📄 **Documentação Técnica:** [Visualizar PDF da Atividade](./Laboratório de Classificação Visual.pdf)

---

## 🚀 O que foi desenvolvido?
*   **Coleta de Dataset:** Organização de amostras de dados (Imagens, Sons ou Posturas).
*   **Treinamento:** Ajuste de épocas e taxas de aprendizado para otimizar o reconhecimento.
*   **Validação:** Testes em tempo real para verificar se a IA classifica corretamente as novas entradas.

## 🛠️ Tecnologias e Ferramentas
*   **Teachable Machine:** Para o motor de Machine Learning.
*   **TensorFlow.js:** Tecnologia base utilizada pelo modelo.
*   **Documentação:** Relatório técnico detalhando os resultados obtidos.

## 📊 Resultados Obtidos
A atividade demonstra a capacidade de implementar soluções de IA de baixo código (No-code/Low-code) e a compreensão de conceitos fundamentais como **Datasets**, **Epochs** e **Confidence Score**.
